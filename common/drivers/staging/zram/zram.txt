Pages written to these disks are compressed and stored</div><div class='line' id='LC10'>in memory itself. These disks allow very fast I/O and compression provides</div><div class='line' id='LC11'>good amounts of memory savings. Default: 1)</div><div class='line' id='LC25'><br/></div><div class='line' id='LC26'>2) Set Disksize (Optional):</div><div class='line' id='LC27'>	Set disk size by writing the value to sysfs node &#39;disksize&#39;</div><div class='line' id='LC28'>	(in bytes). If disksize is not given, default value of 25%</div><div class='line' id='LC29'>	of RAM is used.</div><div class='line' id='LC30'><br/></div><div class='line' id='LC31'>	# Initialize /dev/zram0 with 50MB disksize</div><div class='line' id='LC32'>	echo $((50*1024*1024)) &gt; /sys/block/zram0/disksize</div><div class='line' id='LC33'><br/></div><div class='line' id='LC34'>	NOTE: disksize cannot be changed if the disk contains any</div><div class='line' id='LC35'>	data.
